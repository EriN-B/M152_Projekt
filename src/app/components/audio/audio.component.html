MP3

"MP3" ist die Abkürzung für "MPEG-1 Audio Layer 3" und gehört in die Familie der Motion Picture Expert Group (MPEG). MP3-Dateien werden mit ".mp3" gekennzeichnet.

Die Musik-Datei wird dabei sehr stark komprimiert, wobei sie nur geringfügig an Klang-Qualität verliert. Im Vergleich zur originalen Musik-Datei kommt eine .mp3-Datei mit nur zehn Prozent der ursprünglichen Dateigröße aus.

Bei der Komprimierung werden einfach bestimmte Töne weggelassen. Das betrifft die Töne, die von lauteren Tönen maskiert, also verdeckt werden. Die Kunst dabei ist, genau diese Töne zu finden. Diese sind für das menschliche Ohr nämlich kaum wahrnehmbar und somit entsteht kaum Qualitätsverlust bei der Komprimierung. Das Verfahren gewährleistet somit gute Qualität bei geringer Datengröße.

Vorteile: weit verbreitet, von vielen Abspiel-Geräten unterstützt, hohe Kompression
Nachteile: ggf. hörbarer Qualitätsverlust, vor allem bei kleinen Bitraten

WAV

Beim WAVE- (oder WAV-)Format handelt es sich um ein Containerformat zur verlustfreien Speicherung von Audiodaten. Es basiert auf dem Resource Interchange File Format (RIFF), das von Microsoft und IBM für das Betriebssystem Windows entwickelt wurde.

Die Dateigrösse einer WAV-Datei ist aus dem Grund grösser als eine MP3-Datei, da sie wie schon erwähnt eine Verlustfreie Kompression (PCM) durchgeführt wird. Dadurch gehen nicht wie beim MP3-Format keine Frequenzen verloren.

Vorteile: Muss beim Bearbeiten nicht kodiert / dekodiert werden
Nachteile: Sehr große Dateien

OOG

OGG ist ein komprimiertes digitales Audiodateiformat, das von anderen auch als OGG Vorbis bezeichnet wird. Dieses Audiodateiformat ist im Vergleich zu MP3 nicht so beliebt, aber in Bezug auf die Größe viel kleiner, da es stärker komprimiert ist.

Das OGG-Format ist ein Open-Source-Audioformat. Somit kann jeder dieses Format verwenden, wenn er möchte. Sie sollten auch wissen, dass die Spotify-Anwendung dieses OGG Vorbis-Format verwendet. Wenn Sie überprüfen möchten, wie gut dieses Format ist, können Sie jeden Spotify-Track anhören.

Wenn es um hohe Qualität geht, gelten unter anderem die Dateien im OGG Vorbis-Format als die beste Qualität.

Vorteile: geringe Dateigröße bei guter Klangqualität, lizenzfrei
Nachteile: von vielen Programmen nicht unterstützt, muss für die Audio-Bearbeitung gewandelt werden
