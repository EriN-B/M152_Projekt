Beschreibung

Mit SVG Animationen kann man die Webseite ganz einfach ein wenig lebendiger gestalten. Für die Animationen benötigt man keine Kenntnisse über Javascript, da die Animationen mit reiniem CSS erstellt werden. In unserem Fall haben wir ein SVG welches so aussieht wie ein Geist von Pac-Man. Mit Hilfe von Keyframes wird das SVG bewegt. Zudem werden den Quadraten im SVG CSS-Klassen zugewiesen, um sie z.b. darzustellen oder du verstecken
